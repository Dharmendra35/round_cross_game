# round_cross_game
A simple and interactive ROUND_CROSS game built using HTML, CSS, and JavaScript, providing users with a smooth and responsive interface to enjoy the classic 2-player game.

Features
 Two-player gameplay

 Displays winning combinations

 Clean and responsive user interface

 Option to restart the game

Tech Stack
 HTML – Structure of the game board

 CSS – Styling and layout for a better user experience

 JavaScript – Game logic, user interactions, and win detection
